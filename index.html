額度限額名詞說明
RPM
Requests Per Minute
每分鐘最多可發送的 API 請求次數限制。
TPM
Tokens Per Minute
每分鐘最多可傳輸的總 Token 數限制（包含輸入與輸出）。
RPD
Requests Per Day
每日最多可發送的 API 總請求次數上限。
