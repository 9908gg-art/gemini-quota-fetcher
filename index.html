🎁 免費模型特別說明 (Free Tier)

Google AI Studio 提供了非常佛心的免費額度層級 (Free Tier)，適合個人開發者、學生或做為產品雛形測試。以下是熱門免費模型的特點：

  • Flash 輕量極速模型
    • 免費額度高：提供較高頻率的 RPM / RPD 限制。
    • 極速輕量：支援多模態輸入，最適合語音輸入、翻譯與日常快速問答。
  • Pro 強大推理模型
    • 功能強大：支援超長上下文視窗，具備強悍推理能力。
    • 適合複雜任務：頻率限制通常較低，最適合複雜分析、邏輯推理與程式碼生成。

RPM
Requests Per Minute
每分鐘最多可發送的 API 請求次數限制。

RPD
Requests Per Day
每日最多可發送的 API 總請求次數上限。
